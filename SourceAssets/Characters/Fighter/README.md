# Fighter source asset handoff

Place editable Fighter source files here (`.blend`, source textures, and export notes).

Before handoff:

- use metric units and apply transforms;
- place the origin at the feet;
- face Unreal +X;
- keep mesh, armature, material, and bone names stable;
- export FBX beside the source file;
- record Blender version, export settings, author, and asset license below.

The gameplay capsule is approximately 68 cm half-height. The target chunky Fighter should read around 150-175 cm tall in Unreal while retaining oversized head, hands, shield, shoulders, and weapon.
