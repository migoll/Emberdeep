# Campground source asset handoff

Place editable campground source files here. Prefer a modular kit over one fused level mesh: ground pieces, paths, tents, campfire, fences, crates, barrels, banners, rocks, and vegetation should remain independently placeable.

Before handoff:

- use metric units and apply transforms;
- use grid-friendly dimensions, with 1 Unreal Unit = 1 centimeter;
- place origins at sensible placement pivots;
- use simple collision proxies where needed;
- keep the restricted dark-fantasy palette and chunky silhouettes;
- export runtime FBX files beside the `.blend` sources;
- record Blender version, export settings, author, and asset license below.
